# JDE Order-to-Cash Automation - Baseline Scripts

This directory contains the first working implementation of the JDE Order-to-Cash automation.

## Version 1.0 - March 8, 2026

**File:** `jde-order-to-cash-v1.ts`

### What It Does

Complete end-to-end automation of the JD Edwards EnterpriseOne Order-to-Cash workflow:

1. ✅ **Login** - Authenticates with demo/demo credentials
2. ✅ **Navigate to Address Book (P01012)** - Opens customer management
3. ✅ **Create Customer** - Fills customer form, captures customer number
4. ✅ **Navigate to Sales Order (P4210)** - Opens sales order entry
5. ✅ **Create Sales Order** - Fills order form, captures order number
6. ✅ **Inventory Inquiry (P41200)** - Checks item availability
7. ✅ **Shipment Confirmation (P4205)** - Confirms shipment
8. ✅ **Generate Invoice (R42800)** - Creates invoice, captures invoice number

- **State Management:** Tracks captured values across steps
